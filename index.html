WORDLE

rules You have to guess a five letter word.
  • If the word that you have entered contains a letter that is present in the correct word the letter turns yellow.
  • If the word that you have entered contains the letter at same position as the correct word does the letter turns green.
  • If the word that you have entered contains a letter that is not present in the correct word the letter turns grey.
